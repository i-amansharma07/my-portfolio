## üîπ What is React?

**React** is a JavaScript library created by Facebook for building user interfaces. It allows you to create reusable UI components and manage state effectively in large web apps.

> React is like the engine of a car ‚Äî it gives you power, but you need to build the rest.

### ‚úÖ Pros of React

- Component-based architecture

---

## üîπ What is Next.js?

**Next.js** is a **React-based framework** built by Vercel. It adds key features like routing, server-side rendering (SSR), static site generation (SSG), and full-stack support.

> Think of Next.js as a fully built car using React as its engine.

### ‚úÖ Pros of Next.js

- File-based routing  
- Built-in SSR/SSG/ISR  
- API routes for backend logic  
- Image optimization  
- SEO-friendly out of the box
